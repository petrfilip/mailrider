# Screenshots

Place screenshots here for documentation.

## Required Screenshot

- **web-ui.png** - Screenshot of the MailRider Web UI showing the email list and email detail view

## How to Take a Screenshot

1. Start MailRider: `docker-compose up -d`
2. Send a few test emails to different addresses
3. Open http://localhost:8082 in your browser
4. Take a screenshot showing:
   - Email list on the left
   - Email detail on the right
   - Dark mode UI (optional but recommended)
5. Save as `web-ui.png` in this directory

## Recommended Screenshot Dimensions

- Width: 1200-1600px
- Height: 700-900px
- Format: PNG (for better quality) or JPG
